# Room Temperature Allocation

## Overview
This is a simple C program that compares the temperatures of two rooms and suggests room allocation for Glen and Mark based on the temperature values entered by the user.

## Features
- Accepts temperatures of two rooms.
- Compares both temperatures.
- Displays suitable room allocation.

## Technologies Used
- C Programming Language
- Standard Input/Output Library (stdio.h)

## How to Run

1. Compile the program:
   gcc room_temperature.c -o room_temperature

2. Run the executable:
   ./room_temperature

## Example

Input:
First room temperature: 25
Second room temperature: 30

Output:
Glen must stay in first room and Mark must stay in second room.
